CV Builder is a modern and interactive **resume / CV creation web application** built using **React, Tailwind CSS, and JavaScript**.  
It allows users to create professional resumes step-by-step with live preview and export them as a PDF.

The entire application runs on the client side and works smoothly on both mobile and desktop devices.

- Profile photo upload support
- Add / edit / delete multiple entries (experience, education, projects, etc.)
- Mobile-first and fully responsive design
- Smooth animations and modern UI
- Export resume as PDF using browser printing / PDF tools
